## 🎓 ОСНОВНЫЕ КОМПОНЕНТЫ

### Аутентификация

```
Пароль → bcrypt (хеш) → Сохраняется в БД
Вход → Проверка хеша → JWT (access + refresh)
```

### Авторизация

```
JWT токен → Decode → Получить user_id
Проверка роли (customer/operator/admin)
Проверка ownership (свой ли это объект?)
```

### Валидация

```
Входные данные → Pydantic → Type check + regex + range
Безопасные данные → Возврат только нужных полей
```

### SQL Security

```
User input → Параметр (не строка)
SQLAlchemy ORM → SQL с ? вместо значений
Защита от инъекций → ✅
```
