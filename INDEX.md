## 📦 КОНФИГУРАЦИЯ (3 ФАЙЛА)

### requirements.txt

**Назначение:** Зависимости Python
**Содержит:**

- FastAPI, SQLAlchemy, PostgreSQL driver
- Pydantic для валидации
- bcrypt, PyJWT для безопасности
- pytest, bandit, pip-audit для тестирования
- Все версии явно зафиксированы (==)

### .env.example

**Назначение:** Шаблон переменных окружения
**Содержит:**

- DATABASE_URL (PostgreSQL)
- SECRET_KEY (заполняется автоматически)
- ALGORITHM (HS256)
- TTL параметры (30 мин, 7 дней)
- LOG_LEVEL

**Назначение:** Исключение файлов из git
**Исключает:**

- .env файлы (безопасность)
- \*.db, **pycache**
- venv, virtualenv
- IDE файлы (.idea, .vscode)

## 📊 КЛЮЧЕВЫЕ ПАРАМЕТРЫ

### API Эндпоинты

- **Аутентификация:** 3 эндпоинта
- **Подписки:** 4 эндпоинта
- **Биллинг:** 3 эндпоинта
- **Всего:** 10 эндпоинтов

### Роли

- customer (клиент)
- operator (сотрудник)
- admin (администратор)

### Таблицы БД

- users
- tariff_plans
- subscriptions
- invoices
- audit_logs

### Безопасность

- bcrypt для пароля
- HS256 для JWT
- Access token TTL: 30 минут
- Refresh token TTL: 7 дней
- Brute-force protection: 5 попыток за 15 минут
